# 🔐 CriptoKids - Site de Design

Um site educacional interativo que ensina **Cifra de César** através de módulos interativos e desafios envolventes!

## 📚 Sobre o CriptoKids

O CriptoKids é uma plataforma educacional inovadora desenvolvida para introduzir conceitos fundamentais de criptografia a estudantes de todas as idades. Com foco na **Cifra de César**, um dos algoritmos de criptografia mais clássicos e didáticos, nosso site oferece uma experiência de aprendizado imersiva e divertida.

### 🎯 Objetivo

Tornar o aprendizado de criptografia acessível, interativo e divertido para crianças e adolescentes, desenvolvendo habilidades de raciocínio lógico e pensamento crítico.

## ✨ Funcionalidades Principais

- 🧩 **Módulos Interativos** - Aprenda cifra de César de forma prática e visual
- 🎮 **Desafios Progressivos** - Complete desafios de dificuldade crescente
- ⚡ **Feedback em Tempo Real** - Receba correções instantâneas das suas respostas
- 🏆 **Sistema de Pontuação** - Acompanhe seu progresso com um sistema de recompensas
- 🎨 **Interface Amigável** - Design intuitivo e atrativo para todas as idades
- 📱 **Responsivo** - Funciona em computadores, tablets e dispositivos móveis

## 🚀 Como Começar

### Pré-requisitos

- Node.js (versão 14 ou superior)
- npm ou yarn

### Instalação

1. Clone o repositório:
```bash
git clone https://github.com/fernando-rgomes/Criptokids_site_design.git
cd Criptokids_site_design
```

2. Instale as dependências:
```bash
npm i
```

3. Inicie o servidor de desenvolvimento:
```bash
npm run dev
```

4. Abra seu navegador e acesse:
```
http://localhost:5173
```

## 💻 Tecnologias Utilizadas

- **TypeScript** (97.5%) - Linguagem de programação tipada
- **CSS** (2.2%) - Estilização e design responsivo
- **Vite** - Ferramenta de build rápida e moderna

## 📖 Como Funciona

### Módulos Interativos
- **Introdução à Cifra de César** - Entenda os conceitos básicos
- **Codificador Visual** - Veja em tempo real como a cifra funciona
- **Decodificador de Mensagens** - Descriptografe mensagens secretas

### Desafios
- **Nível Iniciante** - Aprenda os fundamentos
- **Nível Intermediário** - Resolva problemas mais complexos
- **Nível Avançado** - Desafie-se com enigmas criptográficos
